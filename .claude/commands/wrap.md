End-of-session wrap up:

1. **Summarize** what was done (check git log and conversation)
2. **Update Task Master** — mark completed tasks as done
3. **Note blockers** or unresolved issues
4. **Identify next task** for the next session
5. **Capture ideas** — if any came up, append to ideas.md

Present as:
```
═══════════════════════════════
SESSION WRAP — [date]
═══════════════════════════════
Done: [what was completed]
Blockers: [any issues]
Next session: [what to start with]
Ideas captured: [count]
═══════════════════════════════
```

This should be the LAST thing run before ending a session.
